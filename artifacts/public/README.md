# Public Artifacts

This directory contains the small, curated artifact set intended for public publication.

Included here:
- selected JSON summaries
- generated Markdown tables
- publication-grade plots
- lightweight evidence snapshots referenced by the docs site

Excluded from this directory:
- raw datasets
- full result dumps
- large logs
- iterative or noisy local experiment outputs

If a figure or summary exists here, it should be reproducible from the scripts in `scripts/`
and the source artifacts described in `docs/results.md`.
